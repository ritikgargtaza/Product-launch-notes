# Template Deployment Folder

This folder shows you the structure for a launch notes deployment.

## Files

- **PRD.md** — Your product requirements document (read fully by Claude)
- **launch-notes.md** — Generated output (created after running `/project:launch-notes`)

## How to Use This Template

1. Copy this folder: `cp -r .template my-feature-name`
2. Edit `PRD.md` with your actual feature details
3. Run `/project:launch-notes` to generate notes
4. Review `launch-notes.md` and share with teams

## Quick Checklist

- [ ] PRD.md filled in with feature details
- [ ] Includes scope, impact areas, and timeline
- [ ] Flags any regulatory/compliance unknowns
- [ ] Quantifies volumes or customer segments where possible
- [ ] `/project:launch-notes` command runs without errors
- [ ] launch-notes.md generated successfully
- [ ] Changes committed to git
